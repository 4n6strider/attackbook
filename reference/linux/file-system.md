# File System

```text
/bin: essential user commands (usr/bin) 
/boot: boot loader 
/dev: device files 
/etc: host configuration 
/home: user home directories 
/lib: shared libraries and kernel modules 
/media: mount for removable media 
/mnt: mount for temporary file systems 
/opt: add-on applications and packages 
/proc: files for processes 
/root: root user home directory 
/sbin: system binaries 
/srv: data for services 
/tmp: temp files and data 
/usr: user binaries 
/var: variable data, logs, apache www 
```
