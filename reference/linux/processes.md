# Processes



```bash
CTL-C          #kill
CTL-Z          #suspend
jobs           #list 
kill %1        #kill by job number 

ps -T
kill [pid] 
kill -9 [pid]  #force

fg             #resume
bg 
&              #run in background 
nohup 
disown 

ps -aux        #all users not tty 

top -u [user]
```
